\section*{InjecAgent --- direct-harm attacker instructions (verbatim from the benchmark)}

\footnotesize
The \textbf{30 unique direct-harm (\texttt{dh}) attacker instructions} ODILE is evaluated against, grouped by harm type. Each is injected into a tool output the agent reads while doing an ordinary read-only task (see the harness exhibit \texttt{injecagent\_harness.pdf} for the model's-eye view); the tool chip after each line is the attacker's target tool. These are nothing like ODILE's AgentDojo training injections (banking money-transfer / data-exfil under an \texttt{<INFORMATION>} wrapper): different domains (IoT, healthcare, finance apps, e-commerce), different tools, different phrasings. Generalizing to them is not memorization --- ODILE holds \texttt{dh\_base} and \texttt{dh\_enhanced} at $\le 0.15\%$ ASR on every backbone.
